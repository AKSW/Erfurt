# Erfurt

Erfurt is a PHP / Zend based Semantic Web Framework for Social Semantic Software.

## Features

* RDF Parser / Serializer
* Storage Abstraction / RDF Store
* Versioning
* SPARQL Query / Update
* Caching
* Plugins / Trigger
* ...
