		listEntries (line 37)
	
 
	
	

Returns an array of versioning entries.


Returns an array of versioning entries. The entries returned can be specified by the $filter parameter which takes an array of key/value pairs. The following keys are possible: 
  • date &ndash; a date string
  • model &ndash; a model URI
  • user &ndash; a user URI
  • resource &ndash; a resource URI
