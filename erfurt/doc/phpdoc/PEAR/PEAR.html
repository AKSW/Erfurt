Class PEAR

Description
Description | Descendents | Methods (details)

Base class for other PEAR classes. emulation of destructors.

Base class for other PEAR classes. Provides rudimentary emulation of destructors. If you want a destructor in your class, inherit PEAR and make a destructor method called _yourclassname (same name as the constructor, but with a "_" prefix). Also, in your constructor you have to call the PEAR constructor: $this->PEAR();. The destructor method will be called without parameters. Note that at in some SAPI implementations (such as Apache), any output during the request shutdown (in which destructors are called) seems to be discarded. If you need to get any debug information from your destructor, use error_log(), syslog() or something similar. IMPORTANT! To use the emulated destructors you need to create the objects by reference: $obj =& new PEAR_child;
