readAll (line 494)

Read until the socket closes, or until there is no more data in the inner PHP buffer. mode we wait for at least 1 byte of data. blocking mode, if there is no data at all to be read, this function will never exit (unless the socket is closed on the remote end).

Read until the socket closes, or until there is no more data in the inner PHP buffer. If the inner buffer is empty, in blocking mode we wait for at least 1 byte of data. Therefore, in blocking mode, if there is no data at all to be read, this function will never exit (unless the socket is closed on the remote end).

  • return: All data until the socket closes, or a PEAR_Error if not connected.
